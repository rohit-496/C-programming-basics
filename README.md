# Project Directory Overview

This directory contains various C and C++ programs, text files, and compiled executables. Below is a summary of the contents:

## Source Code Files

- **calculator.cpp**: C++ program for basic calculator operations.
- **exam.c**: C program related to exam functionality.
- **fibo.c**: C program to generate Fibonacci numbers.
- **filee.c**, **fileh.c**, **filehu.c**: C programs for file handling operations.
- **HCF.cpp**: C++ program to calculate Highest Common Factor.
- **palindrome.c**, **palindrome.cpp**: Programs to check for palindromes in C and C++.
- **prime.cpp**: C++ program to check for prime numbers.
- **structure.c**: C program demonstrating structures.
- **tempCodeRunnerFile.c**: Temporary file for code execution.

## Executable Files

- **a.exe**, **calculator.exe**, **exam.exe**, **fibo.exe**, **filee.exe**, **fileh.exe**, **filehu.exe**, **HCF.exe**, **palindrome.exe**, **prime.exe**, **structure.exe**, **tempCodeRunnerFile.exe**: Compiled executables for the respective source files.

## Text Files

- **ahaa.txt**, **detination.txt**, **haha.txt**, **paste.txt**, **rohitey.txt**, **text.txt**: Various text files, possibly used for input/output or testing.

## Other

- **README.md**: This documentation file.
- **.git/**: Git repository folder for version control.
